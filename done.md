## Vorlage für erledigte Einträge

Sobald ein Modul abgeschlossen ist, wird es hier mit folgendem Format eingetragen:

```
## [x] Modul N: Name
**Abgeschlossen am:** YYYY-MM-DD  
**Beschreibung:** Was wurde umgesetzt.  
**Kommuniziert mit:** Welche anderen Module / wie (REST, WebSocket, direkt, etc.)  
**Notizen:** Besonderheiten, bekannte Einschränkungen, Follow-ups.
```
